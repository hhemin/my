# 机器人管理后台

负责整个机器人管理后台项目的开发。在该项目中，我采用了高效的脚本模板生成通用代码，以确保代码风格的统一。同时，使用Vue 3进行多语言开发，并与后端进行沟通以统一代码风格。在项目中，我处理了时区、接口返回数据格式和字典等方面的问题。

主要技术涉及：

  1. WebSocket通信：使用WebSocket实现Web与机器人之间的通信。
  2. 视频播放：通过H264进行Web解码播放，支持多个视频源的切换。
  3. 热成像视频展示：利用Base64图片数据和坐标信息展示温度。在处理过程中，我成功解决了导致页面卡顿的问题。

该项目是一个跨团队协作的项目，我与机器人开发人员、安卓开发人员和后台人员等多个团队成员进行了紧密的沟通和协作。我们共同努力解决了每个团队面临的难题，并成功完成了项目目标。

# ETC 智慧出行管理后台（目前公司项目）

ETC 智慧出行管理后台，主要涉及技术点 element-ui、vue 全家桶技术，管理后台分人员管理、商户管理等等，解决不同用 户登录访问页面权限；同时负责有小程序原生开发，参与了开发小程序的无感支付开通等等。开发过程需要同后端人员协同合作，例如需要统一返回字段格式操作等等。

  1. 独立完成前端架构搭建，使用Vue-Admin-Element进行二次开发
  2. 采用Vue 3和TypeScript编写代码，结合当前业务逻辑进行改造。
  3. 实现了人员管理、商户管理等功能，并解决了不同用户登录访问页面权限的问题。
  4. 使用Hook封装模式，如列表加载、表单提交、选项字典等，提高了代码的复用性和开发效率；通过 HBS 文件生成通用的代码（例 如列表、表单），能高效提升开发速度。

# 中山边检智慧后勤平台(微信小程序）

个人对接“中山日报”团队技术开发团队的项目，一个人负责采用原生微信语法和Vant作为UI框架，从零开始搭建了小程序的主体架构。其中主要业务逻辑主要是动态表单提交和一些通知信息

  1. 网络请求模块封装：我对网络请求模块进行了封装和处理，以实现更优雅的后端API调用。例如，通过参数的配置，我能够决定是否需要在调用接口时显示loading提示等功能。
  2. 通用组件开发：在项目中，我开发了许多通用组件，其中对我印象最深刻的是表单组件。该组件集成了多种类型的表单控件，通过参数类型的判断来渲染展示对应的输入框、图片选择器、日期选择器（包括日期和时间选择）、选择器等。我还解决了表单验证是否为空的难点问题，以确保数据的完整性和准确性。
  3. 适配和交流：在项目中，我与后端人员密切合作，采用快捷方便的方式进行适配和交流。我们共同解决了一些接口返回数据格式等问题，确保前后端数据的准确传递和处理。
