# 项目部分图页面

# 1、货币网 (opens new window)未完成，目前还在进度中

可以查看比特币等等相关，查看K线图、货币实时走线图走向、在线卖出买入的情况、同时可以参与买入卖出的网站。后端框架 采用SpringBoot+mysql开发，货币数据来源于第三方api；后台页面采用antd-vue UI框架搭建，前端开发人员3人。 职责：参与前端web页面前台页面开发，负责了前端前台项目页面架构搭建；登录模块；首页数据列表模块（其中参与封装了 列表模块组件）；货币详情页面模块开发（涉及到K线图，走势图和卖出买入情况），k线图和走势图采用WebSocket进行连接 请求格式化数据转换绑定数据。 采用技术点

  • WebSocket、v-echart（用vue进行echart二次封装插件）图表为基础可视化操作；
  • 采用nuxt进行开发。
  • 前端前台BLUMA UI框架
  • 页面采用自适应布局，能适应到手机4.0尺、平板ipad mini 和pro款和电脑pc端。

# 2、数据可视化(未开源)

  • 一个大屏幕展开的查看App下载地区，服务器异常类等等
  • 这个项目采用echart.js 作为图形化(地图，扇形，环型等等可视化)
  • 采用vue-antd UI 框架
  • 采用组件化，地图一个组件，扇形一个组件等等，最后合拼一个页面
  • 采用eslint 进行代码风格管理

# 3、智能识别微信小程序App(未开源)

负责微信小程序app总体架构搭建+数据逻辑绑定，前后端分离合作开发模式。

  • 核心功能通过百度api识别物体，进行判断调用后端api加载物体介绍；
  • 封装微信请求wx.request,避免项目过多出现这些字眼，优雅管理开发
  • 相机界面也进行美化处理等等
  • 这个小程序采取官方原生代码进行开发，同时使用es6语法
  • 采用微信官方的代码管理进行代码管理

# 4、“自游鸟”（公司项目）

是一个酒店预定和发布圈子的项目； 采用单纯原生css+jq写的html页面，不使用UI框架的项目，设计图纸根据UI设计图还原的项目，产品经理的项目要求进行修改开发。页面html大约有统计50个页面。

# 5、 时间管理小程序app (opens new window)

这个项目主要是用来时间计划，可以进行添加、删除、编辑和运行计划，运行计划结束能进行音乐播放主要的业务；要注册才能访问使用，路由方面也采 用路由拦截；登录方面采用token识别当前用户等等

  • 采用mysql+koa写后台接口
  • 前端页面采用Kbone框架
  • Nginx部署代理,采用https
