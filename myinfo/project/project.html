# 项目部分图页面

# 1、ETC智慧出行管理后台（目前公司项目）

ETC智慧出行管理后台，主要涉及技术点element-ui、vue全家桶技术，管理后台分人员管理、商户管理等等，解决不同用 户登录访问页面权限；同时负责有小程序原生开发，参与了开发小程序的无感支付开通等等。开发过程 需要同后端人员协同合作，例如需要统一返回字段格式操作等等。

  • 主要涉及技术element-ui、vue、typescript、微信小程序、vant UI等等。
  • 代码采用eslint格式检查。
  • 模块化开发封装通用代码（例如列表数据加载、分页等等逻辑），通过HBS文件生成通用的代码（例 如列表、表单），能高效提升开发速度。

# 2、乐加油小程序（目前公司项目）

小程序端采用微信小程序原生代码开发，vant为UI主题

### **3、数据可视化(未开源)** * 一个大屏幕展开的查看App下载地区，服务器异常类等等 * 这个项目采用echart.js 作为图形化(地图，扇形，环型等等可视化) * 采用vue-antd UI 框架 * 采用组件化，地图一个组件，扇形一个组件等等，最后合拼一个页面 * 采用eslint 进行代码风格管理

# 4、智能识别微信小程序App(未开源)

负责微信小程序app总体架构搭建+数据逻辑绑定，前后端分离合作开发模式。

  • 核心功能通过百度api识别物体，进行判断调用后端api加载物体介绍；
  • 封装微信请求wx.request,避免项目过多出现这些字眼，优雅管理开发
  • 相机界面也进行美化处理等等
  • 这个小程序采取官方原生代码进行开发，同时使用es6语法
  • 采用微信官方的代码管理进行代码管理

# 5、“自游鸟”（2017年公司项目）

是一个酒店预定和发布圈子的项目； 采用单纯原生css+jq写的html页面，不使用UI框架的项目，设计图纸根据UI设计图还原的项目，产品经理的项目要求进行修改开发。页面html大约有统计50个页面。

# 6、 时间管理小程序app (opens new window)

这个项目主要是用来时间计划，可以进行添加、删除、编辑和运行计划，运行计划结束能进行音乐播放主要的业务；要注册才能访问使用，路由方面也采 用路由拦截；登录方面采用token识别当前用户等等

  • 采用mysql+koa写后台接口
  • 前端页面采用Kbone框架
  • Nginx部署代理,采用https
