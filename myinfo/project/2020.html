# 📱 智能识别微信小程序 App（未开源）

项目简介
负责微信小程序 App 总体架构搭建与数据逻辑绑定，采用前后端分离合作开发模式。成果已发表为 知网论文 (opens new window)。

主要技术亮点

  • 核心功能通过集成百度 API 智能识别物体，判断识别结果并调用后端 API 动态加载物体介绍。
  • 深度封装微信官方请求方法 wx.request，统一管理 API，避免项目中重复出现冗余代码，提升维护效率。
  • 对小程序相机拍摄界面进行了专门的 UI 美化与交互适配。
  • 小程序完全使用官方原生语言进行开发，并全面应用 ES6+ 新语法标准。
  • 采用微信官方代码管理工具进行团队协同开发与版本控制。
