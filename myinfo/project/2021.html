# ETC 智慧出行管理后台

项目简介
ETC 智慧出行管理后台主要涉及 element-ui、vue 全家桶技术。管理后台分人员管理、商户管理等，解决不同用户登录访问页面的权限问题；同时负责小程序原生开发，参与了开发小程序的无感支付开通等。开发过程需要同后端人员协同合作，例如需要统一返回字段格式操作等。

主要工作内容

  1. 独立完成前端架构搭建，使用 Vue-Admin-Element 进行二次开发。
  2. 采用 Vue 3 和 TypeScript 编写代码，结合当前业务逻辑进行改造。
  3. 实现了人员管理、商户管理等功能，并解决了不同用户登录访问页面权限的问题。
  4. 使用 Hook 封装模式，如列表加载、表单提交、选项字典等，提高了代码的复用性和开发效率；通过 HBS 文件生成通用的代码（例如列表、表单），高效提升开发速度。

# 📱 时间管理小程序 app (opens new window)

项目简介
这个项目主要是用来时间计划，可以进行添加、删除、编辑和运行计划，运行计划结束能进行音乐播放主要的业务；要注册才能访问使用，路由方面也采用路由拦截；登录方面采用 token 识别当前用户等。

主要技术点

  • 采用 MySQL + Koa 编写后台接口
  • 前端页面采用 Kbone 框架
  • Nginx 部署代理，启用 HTTPS 安全加密
